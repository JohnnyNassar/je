Infrastructure & security

  • Contabo Cloud VPS 10 — 4 vCPU / 8 GB / NVMe
  • Ubuntu 24.04 LTS, Asia/Amman timezone
  • Nginx + PHP 8.3-FPM + MariaDB 10.11 + Redis 7
  • HTTPS via Let's Encrypt (auto-renew via certbot.timer)
  • DNS at Namecheap → both @ and www → IP
  • SSH: ed25519 key-only, password auth disabled
  • UFW firewall: only 22 / 80 / 443
  • fail2ban active on SSH
  • Daily DB backup at 03:00, 14-day retention
  • Web DB admin at /_adminer (basic auth)
